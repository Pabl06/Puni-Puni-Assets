# Goku Yo-kai
Goku Yo-kai are a set of 5 Yo-kai introduced in Yo-kai Watch: Wibble Wobble. Translated to English as "Extreme Yo-kai," they are SSS Rank Yo-kai of various tribes under the control of Rinne.
